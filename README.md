# blog
source of my personal blog site

### Create a new project
```shell
 > hugo new site ./project_directory
```

### Create a custom theme
```shell
 > hugo new theme mytheme
```

### Create new post
```shell
 > hugo new post_page.md
```

### Run on local server
```shell
 > hugo server --theme=mytheme --buildDrafts
```

### Packaging
```shell
 > hugo
```
